🖥️ שלב 1: הפעלת השרת

השרת הוא הגב של המערכת. הוא מנהל את כל הנתונים, הקיוסקים, המשתמשים וההרשאות.

צעדים:

  1. פתח PowerShell או Command Prompt
  2. עבור לתיקיית הפרויקט:
    cd "c:\Production apps\Claude code\Kiosk_project_Baseline_v2"
  3. הפעל את השרת:
    node server.js
✓ אם הכל תקין:
תראה את ההודעה: Kiosk Server is running on port 5190...
⚠️ אם יש שגיאה:
אם יש שגיאת EADDRINUSE, זה אומר שהפורט כבר תפוס. הרוג את התהליך הקודם עם Ctrl+C

פרטים חשובים:

פרט ערך
כתובת http://localhost:5190
בסיס נתונים C:\SQL_DB\Kiosk_project_Baseline_v2\kiosk.db
הגדרות settings.json בתיקיית הרוט

👨‍💼 שלב 2: ממשק הניהול (Admin Console)

ממשק הניהול שלך מאפשר לך להגדיר קיוסקים, משתמשים, תוכן וטיקטים.

גישה לממשק:

  1. פתח דפדפן אינטרנט
  2. כנס לכתובת:
    http://localhost:5190
  3. התחבר עם שם משתמש וסיסמה:
    שם משתמש סיסמה
    kioskadmin qw12!@

מה אתה יכול לעשות:

  • ➕ הוספת קיוסקים: הגדר מחשבים חדשים עם שם, IP, תיאור
  • 👥 ניהול משתמשים: צור משתמשים חדשים והתן להם הרשאות לקיוסקים
  • 🔗 קישורים: הוסף URL או תמונות שיוצגו בקיוסקים
  • 📢 הודעות: שלח הודעות למשתמשים בקיוסקים
  • ⚙️ הגדרות: שנה נתיבי בסיס נתונים, הגדר כללים
⚠️ חשוב:
שנה את הסיסמה של kioskadmin בהקדם! לא משאיר את הסיסמה הדיפולטית בייצור.

🖱️ שלב 3: הפעלת צד הלקוח - EXE

ה-EXE הוא אפליקציית Electron שמריצה את הקיוסק במסך מלא. הוא מוצא אוטומטי את הקיוסק לפי שם המחשב.

דרישות:

  • Windows 7 ומעלה
  • חיבור לשרת (אותה רשת או IP שניתן להגיע אליו)
  • שם המחשב צריך להתאים לשם בממשק הניהול

הפעלה:

  1. עבור לתיקיה:
    C:\Production apps\Claude code\Kiosk_project_Baseline_v2\PlayerApp
  2. הפעל את הקובץ:
    KioskPlayer.exe

צעדי ההפעלה הראשונה:

  1. כנס את כתובת השרת (URL):
    http://localhost:5190

    או כתובת ה-IP של השרת אם הוא במחשב אחר

  2. המערכת תחפש את הקיוסק לפי שם המחשב של Windows
  3. תוכן הקיוסק יוצג בתצוגה מלאה

קיצורי מקלדת:

קיצור פעולה
ESC יציאה מתצוגה מלאה
F12 איפוס הגדרות וחזרה לצעד ההגדרה
💡 טיפ: אם אתה רוצה להפעיל את זה בכל פעם שהמחשב מתעורר, הוסף קיצור ל-KioskPlayer.exe לתיקיית ההפעלה (Startup).

🌐 שלב 4: הפעלת צד הלקוח - HTML

קובץ HTML בדפדפן - אלטרנטיבה ל-EXE אם אתה רוצה להשתמש בדפדפן.

גישה:

פתח את ה-URL בדפדפן עם מזהה קיוסק:

http://localhost:5190/Player_HTML.html?id=1

הערות:

  • הוחלף את 1 עם מזהה הקיוסק בפועל
  • ניתן להשתמש גם בכתובת IP:
    http://192.168.1.100:5190/Player_HTML.html?id=1

דוגמאות:

קיוסק URL
קיוסק 1 http://localhost:5190/Player_HTML.html?id=1
קיוסק 2 http://localhost:5190/Player_HTML.html?id=2
קיוסק 3 http://localhost:5190/Player_HTML.html?id=3
⚠️ חשוב:
את צריך תמיד את מזהה הקיוסק (?id=X) בכתובת! בלעדיו, הקובץ לא יעבוד.

⚙️ שלב 5: הגדרה ראשונית

1️⃣ הוסף קיוסקים

  1. בממשק הניהול, לחץ על "קיוסקים"
  2. הוסף קיוסק חדש עם פרטים:
    • שם: שם המחשב של Windows (חשוב!)
    • IP: כתובת IP של המחשב (אופציונלי)
    • תיאור: תיאור של המוקד
    • אזור: בחר אזור (צריך להוסיף אזור קודם)

2️⃣ צור אזורים

  1. בממשק הניהול, לחץ על "אזורים"
  2. הוסף אזורים חדשים (למשל: "כניסה", "מזומנים", "בנק")
  3. הקצה קיוסקים לאזורים

3️⃣ צור משתמשים וקבע הרשאות

  1. בממשק הניהול, לחץ על "משתמשים"
  2. הוסף משתמש חדש
  3. בחר את הרשאה:
    • Admin: גישה מלאה לכל הקיוסקים
    • User: גישה רק לאזורים שמוקצים
  4. הקצה אזורים למשתמש

4️⃣ הוסף קישורים ותמונות

  1. בממשק הניהול, בחר קיוסק
  2. לחץ על "קישורים"
  3. הוסף URL או תמונה שתוצג בקיוסק
✓ עכשיו:
המערכת מוכנה! הפעל את ה-EXE או ה-HTML והקיוסק יציג את התוכן שהגדרת.

🔧 פתרון בעיות

❌ בעיה: "לא צוין מזהה קיוסק"

  • סיבה: נפתח את Player_HTML.html בלי ?id=X
  • פתרון: הוסף ?id=1 לכתובת

❌ בעיה: "משתמש לא נמצא"

  • סיבה: שם המחשב לא קיים בממשק הניהול
  • פתרון: הוסף קיוסק עם שם המחשב הנכון

❌ בעיה: "שגיאת חיבור"

  • סיבה: כתובת השרת שגויה או השרת לא פעיל
  • פתרון:
    1. וודא שהשרת עובד (יראה "Kiosk Server is running...")
    2. בדוק שכתובת השרת נכונה
    3. בדוק חיבור רשת (ping localhost)

❌ בעיה: "כתובת כבר בשימוש (EADDRINUSE)"

  • סיבה: תהליך אחר משתמש בפורט 5190
  • פתרון:
    1. תרוג את ה-PowerShell (Ctrl+C)
    2. אם זה לא עוזר, הפעל:
      netstat -ano | findstr :5190
      taskkill /PID [PID] /F

❌ בעיה: EXE לא משתמש בקיוסק הנכון

  • סיבה: שם המחשב לא תואם בדיוק
  • פתרון:
    1. בדוק את שם המחשב שלך בפקודה:
      echo %COMPUTERNAME%
    2. וודא שהקיוסק בממשק הניהול בעל אותו שם בדיוק (כולל אותיות גדולות/קטנות)
